    Privacy architecture

    Built for GitHub Pages and local-first use

    Photos are temporary

    Images are displayed using local browser object URLs and are not stored by this app. Refreshing clears the working session.

    No developer API key

    Do not place a secret key in a public web app. Future AI rendering should use local WebGPU models or a user-provided key.

    Open source friendly

    The project is structured as a static site so it can be hosted on GitHub Pages under the MIT license.
