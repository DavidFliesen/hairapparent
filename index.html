    Privacy model

    Designed for GitHub Pages

    No user images are uploaded, stored, or sent to a server in this static version. Everything is handled in the browser session.

    Local photos

    The app previews selected files using browser object URLs. Reloading the page clears them.

    No hidden key

    A public static website should never contain the developer’s private AI API key.

    Honest AI path

    Photoreal generative previews can be added later through local WebGPU models, a desktop companion, or user-provided keys.
